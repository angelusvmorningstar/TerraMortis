# Story 1d.2: Integrate Editor views into SPA

**As a** developer,
**I want** all Editor functionality (list, sheet, edit) working inside the unified SPA,
**So that** character management works seamlessly alongside the Suite.

## Acceptance Criteria

**Given** the unified HTML shell exists
**When** Editor integration is complete
**Then** character list view renders with filters and search
**And** clicking a character opens the sheet view
**And** edit mode toggles inline on the sheet
**And** save persists to localStorage in v2 format
**And** the Edit tab (identity, attributes/skills) works
