# Story 1a.6: Verify and deploy

**As a** developer,
**I want** to verify all extracted code works identically and deploys to GitHub Pages,
**So that** sub-phase 1a is complete and the app is live.

## Acceptance Criteria

**Given** all CSS, JS, and data have been extracted from both HTML files
**When** the app is loaded via Live Server locally
**Then** all 6 test characters render correctly in the editor list view
**And** each test character's sheet view matches the current output (attributes, skills, merits, disciplines, powers)
**And** edit mode works: fields update, dirty indicator shows, save persists to localStorage
**And** Roll tab constructs correct pools for each test character
**And** dice rolling produces results with correct 10-again/9-again/rote behaviour
**And** Territory tab renders (still using React for now)
**And** Tracker tab stores and retrieves per-character data
**And** no console errors in Chrome or Safari
**And** `git push` triggers GitHub Actions and the app deploys successfully to GitHub Pages
**And** the deployed app works identically to the local version
