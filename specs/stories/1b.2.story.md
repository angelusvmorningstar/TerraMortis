# Story 1b.2: Replace hardcoded colours with CSS variables

**As a** developer,
**I want** every hardcoded colour value outside `theme.css` replaced with a CSS variable reference,
**So that** changing a colour in `theme.css` updates all views consistently.

## Acceptance Criteria

**Given** `css/theme.css` defines all design tokens
**When** the replacement is complete
**Then** no hex, rgb, rgba, or hsl colour values appear in `layout.css`, `editor.css`, `suite.css`, or `components.css` (except `transparent`, `inherit`, `currentColor`)
**And** each hardcoded colour is mapped to an existing or new `--var` in `theme.css`
**And** both apps render identically to before the change
