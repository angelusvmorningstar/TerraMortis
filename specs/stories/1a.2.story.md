# Story 1a.2: Extract CSS from index.html (ST Suite)

**As a** developer,
**I want** all CSS from `index.html` extracted into separate CSS files,
**So that** Suite styling is maintainable and separate from structure.

## Acceptance Criteria

**Given** `public/index.html` currently has a `<style>` block (lines 11-562)
**When** the extraction is complete
**Then** the `<style>` block is replaced with `<link>` tags to external CSS files
**And** CSS is split into: `css/suite.css` (suite-specific styles for Roll, Sheet, Territory, Tracker tabs)
**And** shared styles (bottom nav, tabs, labels, dots) go into `css/components.css` (same file as Story 1a.1)
**And** `:root` design tokens remain in the CSS for now (sub-phase 1b)
**And** the Suite renders identically to the current version on iPad Safari and mobile Chrome
**And** no CSS file exceeds 500 lines
