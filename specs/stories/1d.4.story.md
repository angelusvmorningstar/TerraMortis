# Story 1d.4: Rewrite Territory tab from React to vanilla JS

**As a** developer,
**I want** the Territory Bids tab rewritten in vanilla JS,
**So that** the codebase has no React dependency and is consistent throughout.

## Acceptance Criteria

**Given** the Territory tab currently uses React via CDN (~248 lines)
**When** the rewrite is complete
**Then** Territory tab renders identically using vanilla JS DOM manipulation
**And** all bid functionality preserved: add/remove bids, backing, regent, resolve/unresolve
**And** React CDN script tags are removed from the HTML
**And** no React or ReactDOM references remain in the codebase
