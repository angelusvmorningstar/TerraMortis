# Story 1b.1: Extract design tokens into css/theme.css

**As a** developer,
**I want** all CSS custom properties consolidated into a single `css/theme.css` file,
**So that** design tokens are defined in one place and both apps share the same theme.

## Acceptance Criteria

**Given** CSS custom properties are currently scattered across `layout.css`, `editor.css`, `suite.css`, and `components.css`
**When** the extraction is complete
**Then** a new `css/theme.css` file contains all `:root` custom property declarations
**And** `theme.css` is loaded first in both `tm_editor.html` and `index.html`
**And** the existing CSS files no longer contain `:root` blocks with custom property definitions
**And** both apps render identically to before the change
