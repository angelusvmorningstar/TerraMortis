# Story 1a.1: Extract CSS from tm_editor.html

**As a** developer,
**I want** all CSS from `tm_editor.html` extracted into separate CSS files,
**So that** styling is maintainable and changes don't require searching through 3,000 lines.

## Acceptance Criteria

**Given** `public/tm_editor.html` currently has a `<style>` block (lines 10-436)
**When** the extraction is complete
**Then** the `<style>` block is replaced with `<link>` tags to external CSS files
**And** CSS is split into logical files: `css/editor.css` (editor-specific styles), `css/components.css` (shared components like cards, buttons, form elements), `css/layout.css` (structural layout: grid, flex, positioning)
**And** `:root` design tokens remain in the CSS for now (they move to `theme.css` in sub-phase 1b)
**And** the editor renders identically to the current version in Chrome and Safari
**And** no CSS file exceeds 500 lines
