# Story 1d.3: Integrate Suite tabs into SPA

**As a** developer,
**I want** all Suite functionality (Roll, Sheet, Tracker) working inside the unified SPA,
**So that** ST tools work seamlessly alongside the Editor.

## Acceptance Criteria

**Given** the unified HTML shell exists
**When** Suite integration is complete
**Then** Roll tab works: character picker, discipline picker, pool construction, dice rolling
**And** Suite Sheet tab works: character select, read-only sheet
**And** Tracker tab works: character select, vitae/wp/influence tracking, feeding test
**And** all data comes from the shared v2 character array
