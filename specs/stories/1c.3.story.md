# Story 1c.3: Verify unified data layer and retire old format

**As a** developer,
**I want** to verify both apps work from a single v2 data source,
**So that** the old format can be retired.

## Acceptance Criteria

**Given** both apps read v2 format
**When** verification is complete
**Then** Editor save writes v2 to tm_chars_db in localStorage
**And** Suite reads from tm_chars_db and renders correctly
**And** "Sync to Suite" is no longer needed (both read same key)
**And** old-format CHARS_DATA is removed from Suite code
**And** Playwright tests pass (34/34)
**And** no console errors in either app
