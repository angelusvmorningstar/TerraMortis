# Story 1d.1: Create unified HTML shell with shared navigation

**As a** developer,
**I want** a single HTML file with a top bar that navigates between Editor and Suite views,
**So that** both apps are accessible without page reloads.

## Acceptance Criteria

**Given** the Editor and Suite are currently separate HTML files
**When** the merge is complete
**Then** a single `index.html` contains all view containers
**And** a top navigation bar switches between: Characters (list), Editor, Sheet, Roll, Tracker, Territory
**And** the app loads character data once and shares it across all views
**And** no page reloads when switching between views
